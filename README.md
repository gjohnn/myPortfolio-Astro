## 🚀 Deployment
This portfolio is deployed using **Vercel**. To deploy your own version:
1. Fork this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Deploy on Vercel:
   ```bash
   vercel
   ```
